# TOOLS.md - Strategy & Setup

## 🛠 Search & Tool Strategy

- **Search First:** For any queries regarding current events (prices, news, social metrics), **MUST use Search Tools**. Do not rely on internal memory.
- **Deep Web Fetch:** Use web fetching tools to analyze deep content for maximum accuracy.
- **Verification:** Verify information from at least 2 independent sources before summarizing.
