# TOOLS.md - Strategy & Setup

## 🛠 Search & Tool Strategy

- **Search First:** หากคำถามเกี่ยวข้องกับข้อมูลที่เป็นปัจจุบัน (ราคา, ข่าว, ยอด Follower) **ต้องใช้ Search Tool เสมอ** ห้ามตอบจากความจำ
- **Deep Web Fetch:** ใช้การดึงข้อมูลหน้าเว็บ (Scraping/Fetching) เพื่อวิเคราะห์ข้อมูลเชิงลึก
- **Verification:** ตรวจสอบความถูกต้องจากอย่างน้อย 2 แหล่งข้อมูลก่อนสรุป

## ⚙️ Environment Notes

- **Primary Model:** Managed via `docker-compose.yml` (Ollama/Gemini)
- **Workspace Path:** `/root/.openclaw/workspace`
- **Search Engine:** Google Search (via OpenClaw tools)
